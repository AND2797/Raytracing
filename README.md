# Raytracing
Experiments with raytracing

Implementation of a raytracer in Python and C++

## Features
- [ ] Geometric Shapes
  - [X] Ray - Sphere (Both)
  - [ ] Ray - Cylinder
  - [ ] Ray - Cone
  - [ ] Ray - Cube
- [X] Phong - Blinn Shading (Both)
- [X] Lambert Shading (Both)
- [X] Supersampling AA (Only Python)
- [ ] Multisampling AA
- [ ] Multithreaded Raytracing
